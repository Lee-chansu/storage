※ json 파일
- JavaScript Object Notation의 약자
- 확장자 : *.json / *.js
- 인터넷의 데이터 교환 표준 중에 하나
- 프로그래밍 언어 상관없이 널리 사용하는 데이터 포맷

- 형식 :  {key :  value}
          {
            name : 'john',
            age : 30
          }

* value값으로 들어갈 수 있는 자료유형

1) 숫자 : 20
2) 문자 : "문자열"    - 큰따옴표
3) boolean : true / false 둘중에 하나만
4) 배열(array) : [ ]
5) 객체(object) : { }
6) 빈 값(Null값) : null

* value에 함수(function)가 들어갈 수 있다.
- 함수 : function () {}
- 명칭 : 메서드
