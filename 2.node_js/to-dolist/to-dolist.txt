개인을 위한 스케줄러 

테이블에서 필요한 데이터
id, password, username, email, title, to-dolist

기능
- 로그인
- 회원가입
- 멤버가 입력한 to-dolist 출력
- 로그인하지 않은경우 to-do-list 대신 로그인 버튼이 나오도록


++ nav.ejs ++
- login 버튼
- logout 버튼
- 어떤 아이디로 입력했는지 볼 수 있도록.


-- index.ejs --
- 자신이 입력한 to-dolist 화면에 출력
- 할 일을 더 추가할 수 있는 insert 버튼
- 완료됐거나 삭제하고 싶은 경우 삭제할 수 있는 delete 버튼
- 이름이나 내용을 바꾸고 싶은 경우 update 버튼
- 비로그인시 to-dolist 대신 로그인을 유도하는 메시지 출력.

 ++ login.ejs ++
- 회원가입 버튼
- 로그인 완료 버튼
- 취소 시 index.ejs로 넘어갈 버튼
- 아이디, 비밀번호, 아이디 형식 ,비밀번호 형식에 따른 에러미시지 출력
